# 网球关键词库

> 从 3 篇日志提取 · 2026-07-08 更新

### 🔴 技术类
| 关键词 | 含义 | 频率 |
| :--- | :--- | :--- |
| `停顿` | 正手跑动到位后，身体站稳、转体完成后再挥拍击球。 | 3 |
| `跑-停-打` | 跑动击球的标准三步节奏，不可省略“停”的环节。 | 2 |
| `最高点击球` | 发球时，在球升至最高点、即将下落的瞬间击打。 | 2 |
| `先拉拍再跑` | 跑动前完成引拍动作，避免跑动中仓促拉拍。 | 1 |
| `从下往上拉` | 反手击球时，下蹲降低起始点，向上向前挥拍制造上旋和过网高度。 | 1 |
| `垫步启动` | 对方击球瞬间做一个小垫步，判断来球方向并启动。 | 1 |

### 🟡 战术类
| 关键词 | 含义 | 频率 |
| :--- | :--- | :--- |
| `回中` | 击球后立即回到底线中点偏左的位置，以覆盖更大的正手位区域。 | 2 |
| `打底角` | 面对软球或跑动差的对手，坚决将球击向两个底线角落，调动对手。 | 2 |
| `先站定` | 跑到位后先站稳，再击球，宁可牺牲一点时间也要保证击球质量。 | 1 |

### 🟢 心态类
| 关键词 | 含义 | 频率 |
| :--- | :--- | :--- |
| `专注当下` | 比赛时只关注“发好这一球/接好这一球”，不思考结果或对手。 | 1 |
| `去恐惧化` | 不因对手性别、战绩、姿势而预设强弱，上场前保持心态平稳。 | 1 |
| `比赛即治疗` | 将比赛视为锻炼心态的“心理疗程”，而非单纯追求胜负。 | 1 |
