# 网球关键词库

> 从 5 篇日志提取 · 2026-07-11 更新

### 🔴 技术类
| 关键词 | 含义 | 频率 |
| :--- | :--- | :--- |
| `跑-停-打` | 正手跑动击球的核心流程，强调“停”的环节，身体站稳后再挥拍。 | 5 |
| `停顿` | 正手跑动到位后，身体站稳、转体前的短暂静止状态。 | 4 |
| `回中` | 击球后立即回到底线中点偏左（左手持拍）的位置，准备下一拍。 | 4 |
| `垫步预判` | 对手击球瞬间，自己做一个小的分腿垫步，同时判断来球方向并启动。 | 3 |
| `最高点击球` | 发球时，在球上升至最高点、即将下落的瞬间击球。 | 2 |
| `主动发力` | 使用海德Speed拍时，不能借力，需要主动向前向上发力推送。 | 2 |
| `先拉拍再跑` | 跑向前场接短球时，应在跑动前或跑动中完成引拍，而不是跑到位置再拉拍。 | 1 |
| `侧身跑动` | 前场接球时，应侧身拉拍跑动，而非正面跑动。 | 1 |
| `从下往上拉` | 反手击球时，下蹲降低重心，从更低的位置启动挥拍，轨迹向上向前。 | 1 |
| `保持距离` | 正反手击球时，身体与球要保持一个合适的距离，不要太近。 | 1 |
| `抛球左前` | 左手发球时，抛球点应更靠左前方，以便身体迎前击球。 | 1 |

### 🟡 战术类
| 关键词 | 含义 | 频率 |
| :--- | :--- | :--- |
| `打底角` | 面对软球或浅球时，坚决将球打向对手的两个底线角落，调动对手。 | 2 |
| `回中点` | 击球后立即回到底线中点，准备下一拍。 | 1 |

### 🟢 心态类
| 关键词 | 含义 | 频率 |
| :--- | :--- | :--- |
| `专注当下` | 比赛中只专注于“发好/接好这一球”，不胡思乱想。 | 1 |
| `去恐惧化` | 不因对手性别、战绩、姿势而预设强弱，上场前保持冷静。 | 1 |
