# 当前训练计划

> 最后更新：2026-07-07
> 基于 2 篇日志分析

### 本周训练重点

1. **正手跑动中停顿专项（P0）**
   - 练习方法：底线中点出发，教练/搭档喂正手位（左侧）深球，强制执行“跑→停→打”三步节奏。停顿时默数“1-2”再挥拍，宁可慢一步也要转体完整。每次击球后立即回中点偏左。
   - 进阶：加入随机正反手喂球，正手位必须停顿，反手位可自然流畅击球。

2. **软球进攻策略训练（P1）**
   - 练习方法：教练/搭档喂浅球（发球线附近），强制打两个底角（左/右），不打中路。每10球一组，记录打底角成功率。目标：80%以上打到底角。
   - 心理提示：机会球不手软，回中路等于送给对手。

3. **发球最高点击球打磨（P2）**
   - 练习方法：对墙或场地，高抛后眼睛盯球，在球升至最高点瞬间击打。用手机慢动作录像，对比击球点位置。每20球一组，记录偏晚次数。

### 比赛备忘
- [ ] 赛前提醒：深呼吸一次，扫描身体，默念“跑-停-打”
- [ ] 赛中检查点：每分结束后问自己“我回中了吗？”“我停顿了吗？”
- [ ] 赛后记录项：正手停顿执行次数、软球打底角成功率、发球击球点质量
