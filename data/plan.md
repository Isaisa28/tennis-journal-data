# 当前训练计划

> 最后更新：2026-07-07  
> 基于 2 篇日志分析

### 本周训练重点

1. **正手跑动停顿 + 完整转体**  
   - 练习方法：底线中点出发，教练或发球机送正手位深球 → 跑到位后默念“停→转→打”，先站定再挥拍，宁可慢一拍也要转体完整  
   - 进阶：加入“回中”指令，击球后立即垫步回中点偏左

2. **回中意识自动化（左手持拍版）**  
   - 练习方法：底线对拉时，每打完一拍后大声喊“回中”，强制自己回到中点偏左位置；用标志盘标记回中点，跑动训练中增加回中环节  
   - 比赛模拟：每分结束后，教练随机送正手/反手球，要求回中后再启动

3. **软球/浅球进攻策略**  
   - 练习方法：教练送浅球（中场球），要求必须打两个底角（左/右），不打中路；先练定点，再练跑动中打底角  
   - 心理训练：每次打底角后自我肯定“这是正确选择”，建立信心

### 比赛备忘
- [ ] 赛前提醒：深呼吸一次，扫描身体，默念“停→转→打”  
- [ ] 赛中检查点：每分结束后检查站位是否回中；正手位来球前确认自己已站定  
- [ ] 赛后记录项：记录正手停顿执行次数、回中次数、软球处理是否打底角
